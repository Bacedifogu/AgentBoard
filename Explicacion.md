AgentBoard

El objetivo de este proyecto es comprender mejor como funcionan los MCP Servers.
Para ello crearé un sistema de tablones con notas con distintos estados como "a hacer" "en curso" "finalizado"
El sistema contará con una gestión de usuarios que podrán asignarse las notas, crear nuevas notas o hacer comentarios
Las notas por ahora serán simples textos, para mantenerlo todo sencillo

El frontend del proyecto será desarrollado en angular
El backend del proyecto en java 25 con springboot
